## Usage
  - Open the app
  - Put the url of your project
  - Choose the device you want
  - Enjoy!


## Developing
  - Install the dependencies with `yarn`
  - Run the app in development mode with `yarn dev`

<sub>the app has hot reload buit-in</sub>
